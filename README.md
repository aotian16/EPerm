# EPerm
![img](https://jitpack.io/v/aotian16/EPerm.svg)

Permission util for android 6.

# Install

## By jitpack.io

```
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```

```
	dependencies {
	        compile 'com.github.aotian16:EPerm:v1.0.0'
	}
```
## Or Download this project and import as a module
